ZheNar API Document

GET*, POST*: need login
data_type: required parameter
data_type: optional parameter

HTTP 状态码

  • 200 正常。
  • 400 请求格式错误。
  • 403 没有权限。（如未登录）
  • 404 请求的资源不存在。

Error Code

若返回的字典含有 error_code，客户端要处理错误，且不应将 error_code 本身暴露给用户。

若同时含有 error，要将 error 的内容显示给用户。

如遇 NotLoggedIn，应将用户跳转至登录界面。

  • str student_name
